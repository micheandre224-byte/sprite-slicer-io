**Slicer.io - Sprite Slicer & Video to Sprite Tool**

**Status:** BETA  
**License:** MIT  
**Tech:** React • TypeScript • Tailwind CSS

Slicer.io é uma ferramenta 100% gratuita e client-side feita para desenvolvedores de jogos e artistas de pixel art. Ela facilita o recorte de sprite sheets complexas (como as de Sonic) e a conversão de vídeos curtos em frames de animação.

**✨ Features Principais**

- Detecção automática de sprites com Smart Split e Refine
- Suporte a upload de imagem ou vídeo curto (MP4/WebM 5-30s)
- Separação automática por linhas com contagem de frames
- Preview em tempo real com controle de velocidade e fundos variados
- Toggle de frames (ativar/desativar), reordenação e onion skin
- Ações em massa: Remover Desativados, Selecionar/Inverter, etc.
- Exportação: GIF (com transparente), Spritesheet (Atlas) e **JSON aprimorado** com `frames`, `pivotX` e `pivotY`
- Salvamento completo de projeto (incluindo a imagem via Base64)
- 100% privado: tudo processado no navegador, sem servidor, sem login, sem anúncios
- Multi-idioma (Português, Inglês, Espanhol)

**How to Use**

1. Faça upload de uma sprite sheet ou vídeo curto.
2. Use o Corte Inteligente ou refine manualmente.
3. Ajuste tolerância, distância de mesclagem e tamanho mínimo.
4. Ative/desative frames, reordene e ajuste a animação.
5. Exporte GIF, Atlas ou JSON pronto para Godot/Unity/Mugen.

**Tech Stack**
- Frontend: React 18, TypeScript, Vite
- Styling: Tailwind CSS
- Animações: Framer Motion
- Processamento: Canvas API + gifenc
- Ícones: Lucide React
