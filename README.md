**Slicer.io** is a high-performance, browser-based tool designed for game developers and pixel artists. It simplifies the process of slicing sprite sheets and converting short video clips into animation frames.

## ✨ Features

- **🤖 Auto-Detection:** Automatically detect and slice sprites from any sheet using advanced pixel analysis.
- **📹 Video to Sprite:** Import MP4/WebM videos (5-30s) and extract frames instantly.
- **⚡ Real-time Preview:** Preview your animations with adjustable speed and background contrast.
- **🎨 Background Controls:** Test your sprites against different backgrounds (Checkerboard, Green Screen, Magenta, etc.) to ensure perfect transparency.
- **🔒 100% Private:** All processing happens locally in your browser. No data is ever uploaded to a server.
- **🌍 Multi-language:** Full support for Portuguese, English, and Spanish.
- **💸 100% Free:** No ads, no tracking, no microtransactions.

## 🚀 How to Use

1. **Upload:** Drag and drop an image or a short video clip.
2. **Slice:** Use the "Auto Detect" feature or manually select regions.
3. **Refine:** Adjust tolerance and merge distance to get the perfect cut.
4. **Preview:** Play the animation and check for any missing pixels using the background switcher.
5. **Export:** Download your animation as a high-quality GIF with optional background color replacement.

## 🛠️ Tech Stack

- **Frontend:** React 18, TypeScript, Vite.
- **Styling:** Tailwind CSS.
- **Animations:** Framer Motion.
- **Processing:** Canvas API, gifenc (GIF encoding).
- **Icons:** Lucide React.
